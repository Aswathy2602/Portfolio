UI/UX Design Workshop

Description

I attended a hands-on UI/UX design workshop on February 23, 2024, which focused on the essential principles of user interface and user experience design.

During the workshop, I learnt about:

  • User research methods to understand target audiences and their needs.
  • Creating user personas to represent different user types.
  • Wireframing techniques to outline layout and functionality of designs.
  • Prototyping interactive designs using Figma, a popular design tool.
  • Best practices for usability testing and gathering feedback to improve designs.

The hands-on activities allowed me to apply these concepts in real-time, leading to the development of my own design projects. This workshop significantly enhanced my skills in creating intuitive and aesthetically pleasing user interfaces, and it reinforced the importance of user-centered design in the development process.
